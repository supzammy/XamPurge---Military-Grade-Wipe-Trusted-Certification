Open-source secure data wiping tool providing tamper-proof certification to enable safe resale, donation, and recycling of IT assets. Focused on privacy, security, and sustainability.



# XamPurge

**XamPurge** is an open-source project dedicated to solving the critical and growing problem of secure data destruction. Many individuals and organizations face challenges in safely erasing sensitive information from devices before resale, donation, or recycling, leading to widespread fears of data theft.

In India alone, unused IT assets worth over ₹50,000 crore remain idle, contributing to a mounting electronic waste crisis due to lack of trusted, reliable data wiping solutions. Existing tools are often unreliable, complex, or provide no verifiable proof of data erasure, discouraging safe device reuse.

**Xampurge** aims to fill this gap by empowering users with a transparent, trustworthy, and verifiable data wiping solution. Our project promotes privacy, sustainability, and a circular economy by enabling devices to be securely and confidently repurposed.

---

## Features

- One-click secure data wiping for various storage types  
- Tamper-proof certificates certifying every wipe  
- Cross-platform support with offline usability  
- Community-driven open-source development
